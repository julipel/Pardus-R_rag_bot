# Portfolio website (branch `cursor/portfolio-site-c948`)

This branch adds a **Next.js 15 portfolio site** in the `portfolio-site/` directory.

## Run locally

```bash
cd portfolio-site
npm install
npm run dev
```

## Contents

- Interactive showcase of all @julipel GitHub projects
- Repository maturity analysis
- Commercial case studies
- **Expertise & tech matrix** section
- **MVP → Production** process section
- **Evolution** story (PEr07 → RAG--ChromaDB-)
- Project detail pages with architecture diagrams
- Static export ready for GitHub Pages / Vercel

See [portfolio-site/README.md](./portfolio-site/README.md) and [portfolio-site/REPOSITORY_ANALYSIS.md](./portfolio-site/REPOSITORY_ANALYSIS.md).
