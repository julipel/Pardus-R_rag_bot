# РЭ: общая информация о системе «Пардус-Р»

## Назначение документа

Общая информация из руководства по эксплуатации.

## Основная информация

Система предназначена для получения рентгеновских изображений головы, грудной клетки, конечностей и позвоночника. Применяется в больницах, поликлиниках, перинатальных центрах, ФАП, скорой помощи, травматологии, педиатрии и нестационарных условиях. Не предназначена для маммографии, рентгеноскопии, томографии и ангиографии. Изображение формируется через цифровой детектор и ПО.

## ОБЩАЯ ИНФОРМАЦИЯ ОБ ИЗДЕЛИИ

## Назначение, показания, противопоказания

Назначение: Система предназначена для получения диагностических рентгеновских изображений человеческого тела: головы, грудной клетки, верхних и нижних конечностей, позвоночника.

Показания к применению: проведение оперативных рентгенодиагностических исследований пациентов при общей диагностике в больницах, поликлиниках, перинатальных центрах, фельдшерско-акушерских пунктах, отделениях скорой неотложной помощи, травматологических и педиатрических отделениях, включая обследование пациентов без сознания, а также в нестационарных условиях в перечисленных областях медицины.

## Противопоказания:

Не предназначен для маммографии, рентгеноскопии, томографии и ангиографии.

Побочные эффекты не выявлены.

Потенциальные пользователи: медицинские работники профильных учреждений, врачи, медицинские сестры. К работе с изделием допускаются лица не моложе 18 лет, прошедшие инструктаж и имеющие квалификационную группу не ниже II согласно правилам технической эксплуатации и правилам технической безопасности, и навыки, необходимые для эксплуатации изделия.

## Область применения.

Система используется в больницах, поликлиниках, перинатальных центрах, фельдшерско-акушерских пунктах, отделениях скорой неотложной помощи, травматологических и педиатрических отделениях, включая обследование пациентов без сознания, а также в нестационарных условиях в перечисленных областях медицины.

## Описание и принцип работы системы

Принцип работы:

Рентгеновский аппарат, в конструкции которого имеется рентгеновская трубка, генерирует рентгеновское излучение. После прохождения рентгеновских лу- чей сквозь тело пациента, рентгеновские лучи преобразуются в цифровом детекторе в электрические сигналы, которые поступают и обрабатываются программным обес- печением детектора для формирования изображения.

Применение изделия допускается в соответствии с СанПиН 2.6.4115-2025, СанПиН 2.6.1.2523-09 (НРБ-99/2009), СанПиН 2.6.1.2612-10 (ОСПОРБ 99/2010).

## ОПИСАНИЕ МЕДИЦИНСКОГО ИЗДЕЛИЯ

Аппарат рентгенодиагностический острофокусный «ПАРДУС-Р» с рентгеновской трубкой

### Кнопка включения питания

Кнопка включения питания изделия расположена в нижней части рукоятки изделия. Включение изделия осуществляется однократным нажатием с фиксацией кнопки. Для выключения изделия нажмите кнопку повторно.

### Кнопка ПУСК

Кнопка ПУСК расположена на лицевой части рукоятки изделия и позволяет запустить генерацию рентгеновского излучения в соответствии с установленным режимом съемки.

### Панель управления

Общий вид панели управления представлен на рис. 2. Панель управления изделия оснащена четырьмя управляющими кнопками и цифровым дисплеем для отображения актуальной информации о работе изделия: выбранный режим съемки, состояние аккумуляторной батареи, текущее состояние изделия.

На панели управления расположены следующие кнопки:

с помощью кнопки производится установка режима съемки, где

«1» – минимальное время экспозиции, а «4» – максимальное время экспозиции;

с помощью кнопки производится установка анодного напряжения;

с помощью кнопки производится выбор множителя времени экспозиции: «мин. доза» – время экспозиции не будет увеличено, «макс. доза» – время экспозиции будет увеличено в 2 раза, «пленка» – время экспозиции будет увеличено в 3 раза;

с помощью кнопки производится подтверждение ввода данных. Значения времени экспозиции (с) соответствующее комбинации выбора

«режим» и «детектор» следующие:

| | Детектор | Детектор | Детектор |
| --- | --- | --- | --- |
| Режим | «мин. доза» | «макс. доза» | «пленка» |
| 1 | 0,1 | 0,2 | 0,3 |
| 2 | 0,15 | 0,3 | 0,45 |
| 3 | 0,2 | 0,4 | 0,6 |
| 4 | 0,3 | 0,6 | 0,9 |

Изделие снабжено звуковой сигнализацией рабочего состояния:

при нажатии на кнопку ПУСК для генерации рентгеновского излучения и успешной экспозиции звучит один короткий сигнал;

если изделие не выходит на режим, звучит серия из десяти коротких звуковых сигналов.

### Кнопка внешнего запуска

Предназначена для дистанционной работы на изделии. Кнопка внешнего запуска дублирует кнопку ПУСК.

### Разъем сетевого зарядного устройства

Разъем сетевого зарядного устройства расположен на задней поверхности изделия и предназначен для подключения сетевого зарядного устройства.

### Разъем кнопки внешнего запуска

Разъем кнопки внешнего запуска расположен на задней поверхности изделия и предназначен для подключения кнопки внешнего запуска.

### Аккумуляторная батарея

Расположена внутри изделия. Заряжается с помощью сетевого зарядного устройства от сети общего назначения 220 В. Демонтаж блока аккумуляторов для замены осуществляется только на уполномоченном представителем предприятии.

### Сетевое зарядное устройство

Предназначено для зарядки аккумуляторной батареи.

### Кнопка внешнего запуска

Предназначена для дистанционной работы аппарата.

### Штатив с подставкой для аппарата

Штатив включает в себя треногу с телескопическими ножками и подставку для аппарата.

### Транспортная сумка

Аппарат и его комплектующие складываются в транспортную сумку, ко- торая является потребительской тарой. Сумка выполнена из ворсового трикотаж- ного полотна, подложкой из синтетического материала и металлической крепежной фурнитуры.

### Детектор рентгеновского излучения плоскопанельный цифровой, модель FRV1417W

Детектор применяется для оцифровки рентгеновских изображений с последующей передачей для рентгеновской диагностики. Изделие может работать как в режиме беспроводного подключения, так и в режиме проводного подключения. Пользователь также может подключить резервный кабель к адаптеру, чтобы обеспечить питание для работы или зарядки аккумулятора. Если адаптер не подключен, изделие будет питаться от собственной батареи питания.
