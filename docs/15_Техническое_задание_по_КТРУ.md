# Техническое задание по КТРУ для закупок «Пардус-Р»

## Назначение документа

Описание объекта закупки и технические характеристики для тендера.

## Основная информация

Ключевые параметры: аппарат портативный 1 шт.; высокочастотный генератор моноблочного типа; максимальное анодное напряжение ≥65 кВ, конкретно 65; максимальный анодный ток ≥0,15 мА, конкретно 0,15; фокусное пятно ≤0,4 мм, конкретно 0,2; экспозиция ≤1 с, конкретно 0,9; вес ≤5 кг, конкретно 2,5; автономная работа; детектор; активная площадь ≥420×350 мм, конкретно 430×350; ноутбук; ОЗУ ≥4 ГБ; накопитель ≥1024 ГБ; Windows 10; русифицированная программа.

## Практическое значение

Помогает готовить закупочную документацию и ответы на тендерные требования.

## Вывод

ТЗ фиксирует параметры, обоснования и конкретные значения для закупки.
